# Data associated with "Evolutionary diversification patterns of ectomycorrhizal fungal lineages in the Neotropics"

Romero-Gutierrez et al., 2025

## Available files

- `emc_biorep.tsv` → The 6,004 fungal ITS sequences from the Neotropics and associated data.

- `romero-gutierrez_2025_Extended_Data.docx` →  Additional document including coverage-based rarefaction curves by lineage and heatmaps of nucleotide diversity for select lineages.
